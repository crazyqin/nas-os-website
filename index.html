常见问题

NAS-OS 支持哪些硬件平台？
NAS-OS 支持主流的 x86_64 架构（AMD64）和 ARM 架构（ARM64、ARMv7）。可以在旧笔记本、树莓派、Orange Pi 等设备上运行。最低配置要求：1GB 内存，8GB 存储空间。
数据安全性如何保障？
NAS-OS 采用 btrfs 文件系统，支持快照、RAID 和数据校验。所有数据存储在本地，不会上传到云端。系统定期自动备份配置，更新失败可自动回滚。
可以商业使用吗？
可以。NAS-OS 采用 MIT 开源协议，允许免费用于个人和商业用途。如果需要技术支持或定制开发，可选择专业支持或企业定制方案。
如何更新系统？
NAS-OS 支持 OTA 在线更新。在 Web 面板中点击"检查更新"即可自动下载并安装最新版本。更新过程会自动备份配置，失败时自动回滚到上一个版本。
支持远程访问吗？
支持。NAS-OS 内置 DDNS 动态域名解析、端口转发和 HTTPS 证书自动申请功能。配置后可通过域名安全访问，无需公网 IP 也可通过内网穿透实现远程访问。
遇到问题如何获取帮助？
可以通过以下方式获取帮助：1) GitHub Issues 提交问题；2) GitHub Discussions 参与讨论；3) 查看官方文档；4) 专业支持用户可享受优先技术支持和专属微信群。
